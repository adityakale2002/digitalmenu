===================================================
    Restaurant Management Software
    Distribution Guide
===================================================

This guide explains how to distribute the software to your clients.

PACKAGE CONTENTS
---------------
1. Software Files
   - backend/ (Backend server files)
   - frontend/ (Frontend application files)
   - docs/ (Documentation files)
   - setup.bat (Installation script)
   - README.txt (User guide)
   - install.bat (Dependency installer)

2. Documentation
   - DISTRIBUTION_GUIDE.txt (This file)
   - README.txt (User guide)
   - TROUBLESHOOTING.txt (Troubleshooting guide)

DISTRIBUTION STEPS
-----------------
1. Prepare the Package
   a. Ensure all files are in the correct folders
   b. Test the setup.bat script
   c. Create a ZIP file of the entire folder

2. Send to Client
   a. Share the ZIP file via secure transfer
   b. Provide installation instructions
   c. Include contact information for support

3. Client Installation
   a. Extract the ZIP file
   b. Run setup.bat as administrator
   c. Follow on-screen instructions

INSTALLATION REQUIREMENTS
------------------------
1. System Requirements
   - Windows 10 or later (64-bit)
   - 4GB RAM minimum
   - 1GB free disk space
   - Internet connection

2. Prerequisites
   - Node.js (v18.0.0 or later)
   - MongoDB Community Edition
   - These will be installed automatically by install.bat
